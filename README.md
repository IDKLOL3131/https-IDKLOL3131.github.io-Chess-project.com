# Equation Randomizer

A fun and interactive system of equations solver built with vanilla JavaScript. Test your algebra skills by solving randomly generated systems of two linear equations!

## Features

- Randomly generated systems of linear equations with integer solutions
- Two separate input fields for x and y values
- Step-by-step elimination method explanation when you get the answer wrong
- Mathematical symbols (Â· for multiplication, fraction bars for division)
- Separate answer box highlighting the correct solution
- Score tracking with percentage
- Beautiful animated rainbow gradient background
- Copy-protection on equations to prevent cheating
- Responsive design that fits in fullscreen mode
- "Next Equation" button disabled until you check your answer

## How to Use

1. Open `https://idklol3131.github.io/https-IDKLOL3131.github.io-Chess-project.com/` in your web browser
2. Solve the system of two equations displayed
3. Enter your x and y values in the input fields
4. Click "Check" or press Enter to submit your answer
5. If wrong, review the step-by-step solution provided
6. Click "Next Equation" to get a new problem
7. Watch your score and percentage improve!
